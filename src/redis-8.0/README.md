**Redis - 6.2**


Introduction

This image is suitable for deploying redis 6.2 managed by supervisord.Supervisor jobs conf files can be kept in the supervisor folder, in the root of the project.

Pre-requisites

Before deploying Mariadb-Master, make sure the Vault application is deployed.

Volumes

It is not mandatory to create volumes unless applications need them, these are optional.

Standard AppZ Volumes are enabled.
